停車點列表檔案
Station List File

記述格式

文字編碼為 UTF-8。

此首行文字必須記載檔案格式。
以「#」（井號）作為該行文字的開端的話將會被無視。
BveTs Station List 1.01
#comment
stationKey, stationName, arrivalTime, depertureTime, stoppageTime, defaultTime, signalFlag, alightingTime, passengers, arrivalSoundKey, depertureSoundKey, doorReopen, stuckInDoor
..., ..., ..., ..., ..., ..., ..., ..., ..., ..., ..., ..., ...

屬性

stationKey: 停車點名稱（可隨意）
stationName: 時間表上顯示的停車點名稱
arrivalTime: 到達時間 (hh:mm:ss) (p: 通過)
depertureTime: 出發時間或經過時間 (hh:mm:ss) (t: 終點站)
stoppageTime 標準停車時間 [s]
defaultTime: 跳轉到此停車點後的預設時間 (hh:mm:ss)
signalFlag: 出發訊號（0: 最高訊號 | 1: 在早於 depertureTime 與 stoppageTime 之時顯示危險訊號）
alightingTime: 上落車時間 [s]
passengers: 出發時的負載率
arrivalSoundKey: 車門打開時播放的聲音 (聲音列表檔案中已定義的 soundKey)
depertureSoundKey: 在早於 depertureTime 與 stoppageTime 之時播放的聲音 (聲音列表檔案中已定義的 soundKey)
doorReopen: 車門重新開啟的機會率
stuckInDoor: 乘客被車門夾住的時間 [s]
