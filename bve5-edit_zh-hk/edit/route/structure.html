Structure List File

記述格式

文字編碼為 UTF-8。

BveTs Structure List 1.00 此首行文字必須記載檔案格式。
#comment 以 ";" （分號）作為該行文字的開端的話將會被無視。
structureKey, structureFilePath 設定物件名稱相對應的物件的相對路徑。
..., ...

例子

BveTs Structure List 1.00
track1,tracks\ballast.x
ground,grounds\grass.x
background,background.x
