聲音列表檔案
Sound List File

記述格式

文字編碼為 UTF-8。

BveTs Sound List 1.00 此首行文字必須記載檔案格式。
#comment 以 ";" （分號）作為該行文字的開端的話將會被無視。
soundKey, soundFilePath, bufferCount
..., ...

屬性

soundKey: 聲音名稱（任意文字）
soundFilePath: 相對應的 Wave 聲音檔案的相對路徑
bufferCount: 緩衝區數目（最多可以同時播放聲音的數目）

設定時留意

在本文件列出的聲音，將會於開始遊戲前載入到記憶體中（此與 BVE4 的做法有分別）。
所以假如使用很長的聲音，或者使用很多緩衝區的話，記憶體耗用量將會十分之大。敬請留意。
