ATS 插件結構

ATS_VEHICLESPEC

列車規格

struct ATS_VEHICLESPEC
{
  int BrakeNotches;
  int PowerNotches;
  int AtsNotch;
  int B67Notch;
  int Cars;
};

成員

BrakeNotches
制動桿格數。
PowerNotches
動力桿格數。
AtsNotch
ATS 確認格數。
B67Notch
制動桿以 67% 制動時所指向的格數。
Cars
車卡數目。

ATS_VEHICLESTATE

列車狀態

struct ATS_VEHICLESTATE
{
  double Location;
  float Speed;
  int Time;
  float BcPressure;
  float MrPressure;
  float ErPressure;
  float BpPressure;
  float SapPressure;
  float Current;
};

成員

Location
列車位置 [m]。
Speed
列車速度 [km/h]。
Time
現在時間 [ms]。
BcPressure
制動缸壓力 [Pa]。
MrPressure
主風缸壓力 [Pa]。
ErPressure
平衡風缸壓力 [Pa]。
BpPressure
制動管路壓力 [Pa]。
SapPressure
直通管壓力 [Pa]。
Current
電流 [A]。

ATS_BEACONDATA

當駛過地上訊號器時列車上所接收到的資訊

struct ATS_BEACONDATA
{
  int Type;
  int Signal;
  float Distance;
  int Optional;
};

成員

Type
地上訊號器類別。
Signal
對應的訊號區間。
Distance
對應的訊號區間的終結的距離 [m]。
Optional
地上訊號器所設定的任意值

ATS_HANDLES

輸出到 Bve trainsim 的逆轉器、動力器及制動器的值。

struct ATS_HANDLES
{
  int Brake;
  int Power;
  int Reverser;
  int ConstantSpeed;
};

成員

Brake
制動格數。
Power
動力格數。
Reverser
逆轉器位置。
ConstantSpeed
定速制御的狀態。以下為可能的值。
ATS_CONSTANTSPEED_ENABLE 起動
ATS_CONSTANTSPEED_DISABLE 停止
ATS_CONSTANTSPEED_CONTINUE 保持現狀
