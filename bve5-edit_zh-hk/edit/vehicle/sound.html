列車聲音檔案
Vehicle Sound File


記述格式

以INI 格式作為基本記述格式。文字編碼為 UTF-8。

Bvets Vehicle Sound 2.0 此首行文字必須記載檔案格式。
[section1] 宣告新部分。部分的名稱以「[ ]」括起。
key1 = path1 設定該部分某聲音項目的 Wave 聲音檔的位置。"=" 的左方為聲音項目名稱，而右方則為 Wave 聲音檔的位置。
key2 = path2
...
;comment 以「;」（分號）作為該行文字的開端的話將會被無視。
[section2] 宣告另一個新部分。
key3 = path3
...

キー


[Run] 部分
車輪滾動聲

屬性名稱可為 0 及以上的整數


[Flange] 部分
行駛曲軌時的磨擦聲

屬性名稱可為 0 及以上的整數


[Motor] 部分
摩打上

屬性名稱可為 0 及以上的整數


[Joint] 部分
經過道岔（波口）軌條間的空隙時，以行駛速度作與聲音播放速度成正比例，播放一次的聲音

屬性名稱可為 0 及以上的整數


[Brake] 部分
制動系統中各部件氣壓變動時產生的聲音

BcReleaseHigh
制動缸從高壓緩解時產生的聲音

BcRelease
制動缸緩解時產生的聲音

BcReleaseFull
制動缸完全緩解時產生的聲音

Emergency
制動管緊急減壓的聲音


[Compressor] 部分
空氣壓縮機運作時產生的聲音

Attack
啟動頭 5 秒的聲音

Loop
運作時的聲音

Release
停止時的聲音


[AirSpring] 部分
懸掛系統中氣袋承壓變動時產生的聲音

LeftRelease
左邊懸掛氣袋的排氣聲

RightRelease
右邊懸掛氣袋的排氣聲

LeftApply
左邊懸掛氣袋的送風聲

RightApply
右邊懸掛氣袋的排氣聲


[Breaker] 部分
斷流器運作的聲音

On
斷路閉合時產生的聲音

Off
斷路斷開時產生的聲音


[Horn] 部分
響鞍

Primary
警笛 1

PrimaryRelease
警笛 1 餘韻

Secondary
警笛 2

SecondaryRelease
警笛 2 餘韻


[Door] 部分
車門開關聲

OpenLeft
左邊車門打開聲

OpenRight
右邊車門打開聲

CloseLeft
左邊車門關上聲

CloseRight
右邊車門關上聲


[Ats] 部分
由 ATS 插件控制其運作的聲音

屬性名稱可為 0 至 255 之間的任何整數


[Buzzer] 部分

Off

Max

Release

Min
動力桿格數到底時的操作聲

Max

Off
