駕駛室面板文件
Instrument Panel File


記述格式

以INI 格式作為基本記述格式。文字編碼為 UTF-8。

Version 1.0 此首行文字必須記載檔案格式。
[section1] 宣告新部分。部分的名稱以「[ ]」括起。
key1 = value1 設定該部分某屬性的值。"=" 的左方為屬性名稱，而右方則為值。
key2 = value2
...
;comment 以「;」（分號）作為該行文字的開端的話將會被無視。
[section2] 宣告另一個新部分。
key3 = value3
...

値


subjectKey

名稱 型態 說明
true
 bool 常為 true
kmph
 float 速度 [km/h]
kmphd0
 int 速度 [km/h] 的個位數
kmphd1
 int 速度 [km/h] 的十位數
kmphd2
 int 速度 [km/h] 的百位數
bc
 float 制動缸氣壓 [kPa]
bcd1
 int 制動缸氣壓 [kPa] 的十位數
bcd2
 int 制動缸氣壓 [kPa] 的百位數
mr
 float 主風缸氣壓 [kPa]
mrd1
 int 主風缸氣壓 [kPa] 的十位數
mrd2
 int 主風缸氣壓 [kPa] 的百位數
sap
 float 直通管氣壓 [kPa]
bp
 float 制動管壓 [kPa]
am
 float 電流 [A] (加速時: +, 減速時: -)
amabs
 float 電流 [A] (無論加速或減速均為: +)
door
 bool 車門關閉燈 (亮燈: true)
doorl0～doorl15
 bool 左車門燈 (亮燈: true, 頭卡: doorl0)
doorr0～doorr15
 bool 右車門燈 (亮燈: true, 頭卡: doorr0)
ats0~ats255
 int ATS 插件所輸出的值
power
 int 動力桿現時位置的格數
brake
 int 制動桿現時位置的格數
hour
 float 小時 [hour]
min
 float 分鐘 [min]
sec
 int 秒數 [sec]

layerIndex

可設定值為 1 至 15 之間的整數。値越大圖片就越在高（靠近自己）的位置顯示。This 的圖片的圖層值為 0 。

angle

原點處為 12 點鐘的方向，正數指向順時針方向。若指針需轉動多於一周，LastAngle 的值請設定為大於 InitialAngle + 360 的值。

path

該檔案的相對路徑。

color

以「#」及一組六位的十六進制數組成的 HEX 顏色代碼（例如：#FFFFFF）。
