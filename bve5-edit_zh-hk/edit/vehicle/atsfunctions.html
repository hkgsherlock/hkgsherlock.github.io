ATS 插件函數

Load

當載入插件時被呼叫的函數。

ATS_API void WINAPI Load();

Dispose

當釋放插件時被呼叫的函數。

ATS_API void WINAPI Dispose();

GetPluginVersion

當插件被主程式查詢其版本時被呼叫的函數，並傳回一整數值到主程式。

ATS_API int WINAPI GetPluginVersion();

傳回値

本插件的版本號。必須傳回 ATS_VERSION 的值。

SetVehicleSpec

當列車載入時被呼叫的函數。

ATS_API void WINAPI SetVehicleSpec( ATS_VEHICLESPEC vehicleSpec );

屬性

vehicleSpec
列車規格。

Initialize

遊戲開始時（或選擇「Reset Scenario」、跳轉車站時）被呼叫的函數。

ATS_API void WINAPI Initialize( int brake );

屬性

brake
當遊戲開始時制動閥的狀態。以下為本函數可能傳入的參數。

ATS_INIT_SVC 常用制動
ATS_INIT_EMG 緊急制動
ATS_INIT_REMOVED 保安裝置未啟動

Elapse

每一幀都會被呼叫的函數。

ATS_API ATS_HANDLES WINAPI Elapse( ATS_VEHICLESTATE vehicleState, int[] panel, int[] sound );

屬性

vehicleState
列車狀態。
panel
顯示於標板組件的值
sound
每個聲音的分貝值除以 100[dB] 後的值。
ATS_SOUND_PLAY 播放一次
ATS_SOUND_PLAYLOOPING 重覆播放
ATS_SOUND_STOP 停止
ATS_SOUND_CONTINUE 維持該聲音的現狀

傳回値

輸出到 Bve trainsim 的逆轉器、動力器及制動器的值。

SetPower

當動力桿被調節時所被呼叫的函數。

ATS_API void WINAPI SetPower( int notch );

屬性

notch
格數。

SetBrake

當制動桿被調節時所被呼叫的函數。

ATS_API void WINAPI SetBrake( int notch );

屬性

notch
格數。

SetReverser

當逆轉器被調節時所被呼叫的函數。

ATS_API void WINAPI SetReverser( int pos );

屬性

pos
逆轉桿位置。

KeyDown

當 ATS 的按紐被按下時所呼叫的函數。

ATS_API void WINAPI KeyDown( int atsKeyCode );

屬性

atsKeyCode
ATS 鍵的代號。以下為本函數可能傳入的參數。

ATS_KEY_S S 鍵（預設為鍵盤上的 Space）
ATS_KEY_A1 A1 鍵（預設為鍵盤上的 Insert）
ATS_KEY_A2 A2 鍵（預設為鍵盤上的 Delete）
ATS_KEY_B1 B1 鍵（預設為鍵盤上的 Home）
ATS_KEY_B2 B2 鍵（預設為鍵盤上的 End）
ATS_KEY_C1 C1 鍵（預設為鍵盤上的 Page Up）
ATS_KEY_C2 C2 鍵（預設為鍵盤上的 Page Down）
ATS_KEY_D D 鍵（預設為鍵盤上的 2）
ATS_KEY_E E 鍵（預設為鍵盤上的 3）
ATS_KEY_F F 鍵（預設為鍵盤上的 4）
ATS_KEY_G G 鍵（預設為鍵盤上的 5）
ATS_KEY_H H 鍵（預設為鍵盤上的 6）
ATS_KEY_I I 鍵（預設為鍵盤上的 7）
ATS_KEY_J J 鍵（預設為鍵盤上的 8）
ATS_KEY_K K 鍵（預設為鍵盤上的 9）
ATS_KEY_L L 鍵（預設為鍵盤上的 0）

HornBlow

當響鞍時所呼叫的函數。

屬性

hornType

ATS_HORN_MUSIC 音樂警笛
